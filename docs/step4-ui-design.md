## 画面一覧

### 1. 資格一覧ページ（`/`）

**目的：** 関心のある資格を探す

```
┌─────────────────────────────────────────────────────┐
│  Navbar: ロゴ | 資格一覧 | カレンダー | 管理画面     │
├─────────────────────────────────────────────────────┤
│                                                      │
│  [検索ボックス: 資格名で検索...]                      │
│                                                      │
│  大カテゴリ: [すべて] [国家資格] [民間資格] [公的資格] │
│  小カテゴリ: [クラウド] [IT・情報] [語学] [会計] ... │
│                                                      │
│  ○件の資格                                           │
│                                                      │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐            │
│  │ 資格カード│ │ 資格カード│ │ 資格カード│            │
│  │          │ │          │ │          │            │
│  │ バッジ   │ │ バッジ   │ │ バッジ   │            │
│  │ 資格名   │ │ 資格名   │ │ 資格名   │            │
│  │ 試験日   │ │ 試験日   │ │ 試験日   │            │
│  │ 受験料   │ │ 受験料   │ │ 受験料   │            │
│  └──────────┘ └──────────┘ └──────────┘            │
│                                                      │
└─────────────────────────────────────────────────────┘
```

**UX要件：**
- 検索はデバウンス300ms
- カテゴリフィルターはURLクエリパラメータに反映（ブックマーク可能）
- カードはホバーで浮き上がるアニメーション
- 試験日が近い資格は強調表示

### 2. 資格詳細ページ（`/qualification/:id`）

**目的：** 資格の詳細情報を確認し、最新情報を取得する

```
┌─────────────────────────────────────────────────────┐
│  ← 戻る                                              │
├─────────────────────────────────────────────────────┤
│                                                      │
│  [バッジ: 民間資格 / クラウド]                        │
│  AWS Certified Solutions Architect - Associate       │
│  資格の説明テキスト...                                │
│                                                      │
│  ┌─────────────────────────────────────────────┐    │
│  │  スケジュール情報                 [最新情報取得] │  │
│  │                                              │    │
│  │  試験日:        随時（テストセンター）        │    │
│  │  申込開始:      随時                         │    │
│  │  申込締切:      随時                         │    │
│  │  合格発表:      即時（CBT）                  │    │
│  │  受験料:        ¥20,000（税込）              │    │
│  │  最終取得:      2026-06-05 14:32             │    │
│  └─────────────────────────────────────────────┘    │
│                                                      │
│  [公式ページを開く →]                                │
│                                                      │
└─────────────────────────────────────────────────────┘
```

**UX要件：**
- 「最新情報取得」ボタンはスクレイパー非対応資格では非表示
- 取得中はボタンをローディング状態に
- 取得成功/失敗のトースト通知

### 3. カレンダーページ（`/calendar`）

**目的：** 試験日程を時系列で把握する

```
┌─────────────────────────────────────────────────────┐
│  カレンダー                                          │
│                                                      │
│  [すべて] [試験日] [申込開始] [申込締切] [合格発表]   │
│                                                      │
│  ● 試験日  ● 申込開始日  ● 申込締切日  ● 合格発表日   │
│                                                      │
│  ┌─────────────────────────────────────────────┐    │
│  │ ← 2026年6月 →                                │    │
│  │  日  月  火  水  木  金  土                  │    │
│  │  ...                                        │    │
│  │  [基本情報試験日]                            │    │
│  │  [AWS SAA 申込締切]                          │    │
│  │  ...                                        │    │
│  └─────────────────────────────────────────────┘    │
│                                                      │
└─────────────────────────────────────────────────────┘
```

**イベントクリック時ポップアップ：**
- 試験種別バッジ
- 資格名
- 日付
- 受験料（あれば）
- 備考（あれば）
- 「詳細を見る →」リンク

---

### 4. 管理画面（`/admin`）

**目的：** 資格マスタとスケジュールの管理

**タブ構成：**
- `資格一覧` — テーブル表示、編集・削除・日程編集ボタン
- `資格追加` — フォーム（名前・カテゴリ・URL・説明）
- `スケジュール編集` — 日付フォーム（手動登録）
- `取得ログ` — 直近100件のスクレイピングログ

**機能：**
- 「全資格の情報を取得」ボタン（一括スクレイピング）
- 各資格行の「編集」「日程編集」「削除」ボタン

## アニメーション方針

| 対象 | アニメーション |
|------|-------------|
| カードホバー | translateY(-2px) + shadow強化、150ms ease |
| ページ遷移 | fadeIn 200ms |
| モーダル表示 | scale 0.95→1 + opacity、150ms ease |
| ボタンクリック | scale 0.97、100ms |
| ローディング | スピナー（Tailwindアニメーション） |
| トースト | slide-in from top-right |

## ガラスモーフィズムの適用箇所

過度な使用を避け、以下の箇所のみに限定する。

- Navbar（スクロール時に `backdrop-blur` + 半透明背景）
- カレンダーイベントポップアップ
- ローディングオーバーレイ

---

## アクセシビリティ要件

- `aria-label` を全インタラクティブ要素に付与
- キーボードナビゲーション対応
- フォーカスリングを明確に表示
- 色だけに依存しない情報伝達（アイコン + テキスト併用）
- コントラスト比 4.5:1 以上（WCAG AA）
